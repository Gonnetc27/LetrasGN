# LetrasGN 🎵

Una aplicación de escritorio flotante y transparente desarrollada en **Electron** que muestra en tiempo real las letras de las canciones que estás escuchando. 

Ideal para tener estilo "karaoke" mientras usás otras aplicaciones de fondo sin interrupciones.

## 🚀 Características
- **Sincronización en tiempo real:** Detecta la música de Spotify, YouTube u otros reproductores usando sesiones de medios de Windows.
- **Diseño minimalista y transparente:** Se superpone de forma sutil y elegante sin bloquear tu pantalla.
- **Always on top:** Permanece siempre visible por encima del resto de tus ventanas.
- **Modo fantasma:** Los clics traspasan la ventana cuando no estás interactuando con los controles.

## 🛠️ Tecnologías utilizadas
- JavaScript
- Electron
- HTML5 / CSS3
- Librería `windows-media-sessions`

## ⚙️ Instalación para desarrollo
1. Cloná este repositorio.
2. Ejecutá `npm install` para descargar las dependencias.
3. Ejecutá `npm start` para levantar la aplicación.
4. Para compilar el ejecutable (.exe), usá `npm run dist`.
